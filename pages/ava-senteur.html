AvA Senteurs

Présentation du projet

AVA Senteur est une plateforme e-commerce spécialisée dans la vente de parfums en ligne avec une interface moderne, élégante et responsive.

Objectifs

  • Créer une boutique en ligne moderne
  • Améliorer l’expérience utilisateur
  • Concevoir un design responsive
  • Mettre en valeur les produits

Technologies utilisées

HTML CSS JavaScript

Fonctionnalités

  • Catalogue de parfums
  • Navigation fluide
  • Design responsive
  • Présentation moderne des produits
